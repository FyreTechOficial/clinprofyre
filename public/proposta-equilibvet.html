Sobre o Projeto

O que vamos fazer pela EquilibVet

Vamos implementar o ClinPRO personalizado para a realidade da EquilibVet — uma clínica veterinária que atende por convênio PetLove e particular, com serviços de acupuntura, fisioterapia e atendimento clínico geral.

O sistema será adaptado com nomenclaturas e fluxos específicos para o universo pet, mantendo toda a potência dos agentes de IA para atendimento, qualificação, confirmação e follow-up via WhatsApp.

O que está incluso

Funcionalidades do ClinPRO para EquilibVet

🤖

Atendimento 24/7 por IA

IA responde WhatsApp em segundos. Tira dúvidas sobre procedimentos, convênio PetLove, valores e horários.

⭐

Qualificação Automática

Identifica interesse (acupuntura, fisio, consulta), classifica urgência e preenche cadastro do tutor + pet.

📅

Confirmação de Consultas

Confirma agendamentos 48h e 24h antes. Reagenda automaticamente. Reduz no-show em até 40%.

⚡

Follow-up Automático

D1, D3, D7 após consulta. Recupera tutores que não retornaram. Reativa inativos há +90 dias.

📊

Dashboard em Tempo Real

Métricas de leads, agendamentos, mensagens dos agentes e funil de conversão. Tudo ao vivo.

🎯

Pipeline de Vendas

Kanban com 8 estágios. A IA move os leads conforme qualifica. Drag & drop para ajuste manual.

🏷️ Nomenclaturas Personalizadas

O sistema será adaptado para o contexto veterinário: "Paciente" → "Pet/Tutor", "Consulta" → "Atendimento", "Procedimento" → "Serviço", entre outros ajustes de linguagem nos agentes, dashboard e pipeline.

Integrações

Conectando com VetSmart e PetLove

Além das funcionalidades core do ClinPRO, vamos trabalhar nas integrações com os sistemas que a EquilibVet já utiliza.

🩺

VetSmart — Prontuário Eletrônico

Integração para enviar dados do atendimento ClinPRO direto para o prontuário no VetSmart. Cadastro do tutor, pet, histórico de interações e procedimentos realizados.

Verificação de API necessária
💙

PetLove — Pesquisa de Convênio

Quando o tutor envia os dados pelo WhatsApp, a IA faz a pesquisa de cobertura no convênio PetLove e retorna a informação ao tutor automaticamente dentro da conversa.

Verificação de API necessária
📅

Agenda ClinPRO ↔ VetSmart

Sincronização bidirecional da agenda. Quando a IA agenda no ClinPRO, reflete no VetSmart — e vice-versa. Sem duplicidade de agendamentos.

Verificação de API necessária
✅

WhatsApp via Evolution API

Conexão direta com o WhatsApp da clínica via QR Code. Sem trocar de número. Atendimento automatizado 24h.

Pronto para uso
⚠️ Nota sobre integrações VetSmart e PetLove: A viabilidade técnica dessas integrações depende da disponibilidade e acesso às APIs desses sistemas. Vamos verificar durante a fase de implementação e, caso não haja API aberta, proporemos alternativas (ex: webhook, importação manual ou integração via n8n).
